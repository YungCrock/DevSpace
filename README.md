Sobre o aplicativo
Este aplicativo foi desenvolvido como parte de uma atividade acadêmica proposta em sala de aula, com o objetivo de aplicar, na prática, os conhecimentos adquiridos durante o estudo de programação e desenvolvimento mobile.
A proposta do projeto é demonstrar a construção de uma aplicação funcional, explorando conceitos como criação de interfaces, navegação entre telas, organização de componentes e aplicação de boas práticas de desenvolvimento. Durante o processo, buscamos não apenas atender aos requisitos da atividade, mas também desenvolver uma solução bem estruturada, intuitiva e visualmente agradável.
Mesmo sendo um projeto acadêmico, este aplicativo representa um passo importante na nossa jornada como desenvolvedores, permitindo colocar em prática habilidades técnicas, lógica de programação e criatividade na resolução de problemas.
Este trabalho reforça a importância do aprendizado prático e evidencia como desafios propostos em ambiente educacional podem contribuir diretamente para a evolução profissional e o desenvolvimento de projetos reais no futuro.

Como Rodar:
Após dar o clone do projeto, instale as dependencias com um ´npm install´ no terminal. Depois digite ´npm run start´ para rodar o expo.
Caso esteja utilizando o app Expo Go, leia o código QR que aparece no terminal ou digite a url.
Mas se estiver utilizando um emulador apenas digite "A" no terminal.
